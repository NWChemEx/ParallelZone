template<typename VariantArchive>
class parallelzone::ArchiveWrapper< VariantArchive >

ArchiveWrapper class enables serialization and deserialization of objects using various types of archives supported by Cereal.

This class wraps Cereal API for different output archive types. Required for serialization of Any.

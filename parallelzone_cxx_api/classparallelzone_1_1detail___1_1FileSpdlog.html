Detailed Description

Logger which writes to file.

This class specializes SpdlogPIMPL for when the sink is a file. Under the hood we use Spdlog to do the actual logging.

Creates a LoggerPIMPL whose sink is the specified file.

The object resulting from this function is setup so that logged records are written directly to the provided file. If the file already exists the resulting logger will append to it.

Parameters
[in]idAn ID for the logger.
[in]file_nameThe path where the file will exist. If file_name is relative it will be created relative to where the program is currently running.
Returns
SpdlogPIMPL wrapping an spdlog::logger setup according to this function's description.

Detailed Description

Implements LoggerPIMPL so the sink is standard out.

This class is more or less a strong type so we can tell the Spdlog backend apart.

Constructor & Destructor Documentation

parallelzone::detail_::StdoutSpdlog::StdoutSpdlog ( const_string_reference  id)
explicit

Creates a LoggerPIMPL whose sink is standard out.

The object resulting from this function is setup so that logged records are directed to standard out, in color, and in a thread-safe manner. Spdlog is used to actually implement the logger.

Parameters
[in]idAn ID for the logger, used to distinguish loggers in the output.

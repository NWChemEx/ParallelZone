Detailed Description

template<typename InternalBuffer>
class parallelzone::mpi_helpers::detail_::BinaryBufferPIMPL< InternalBuffer >

Implements base's API for type InternalBuffer.

The BinaryBufferPIMPL class implements BinaryBufferPIMPLBase by wrapping a contiguous buffer. Any contiguous container which is relatively standards compliant should work (see description of InteranlBuffer for the exact requirements).

Template Parameters
InternalBufferThe container used to actually store the binary data. InternalBuffer is assumed to have data() and size() methods as well as a member value_type which is a typedef describing the type of the actual buffer elements.

Creates *this by taking ownership of buffer.

BinaryBuffer owns the binary buffer inside it. This ctor is used to initialize that buffer. By default this ctor will initialize buffer by copying the argument provided to the ctor. To avoid the copy you must move the buffer into the ctor.

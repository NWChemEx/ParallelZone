Detailed Description

Holds the state for the RuntimeView class.

This class uses CommPP to manage MPI.

bool parallelzone::runtime::detail_::RuntimeViewPIMPL::operator== ( const RuntimeViewPIMPL &  rhs) const
inlinenoexcept

Determines if *this is value equal to rhs.

Two RuntimeViewPIMPL instances are value equal if they both wrap the same MPI communicator and if they both have access to the same resource sets. At the moment the content of the resource sets is populated off of the MPI communicator so we only compare the MPI communicators.

Eventually the user will be able to logically partition the RuntimeView and at this time operator== will need revisited.

N.B. This class only defines operator== because that is all that is needed for RuntimeView to define both operator== and operator!=.

Parameters
[in]rhsThe RuntimeView we are comparing to.
Returns
True if *this is value equal to rhs and false otherwise.
Exceptions
NoneNo throw guarantee.

Detailed Description

template<typename T>
struct parallelzone::task::ArgumentTraits< T >

Structure that works out cv-qualifiers and reference-ness of T.

Template Parameters
Thetype being inspected.

This structure is predominantly intended for use in template contexts where the properties of a type are not known. In those contexts, it can be useful to know if a type T is a reference, a const reference etc.
